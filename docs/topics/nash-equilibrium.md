# Nash equilibrium

**Idea:** A strategy profile where no player can improve by deviating unilaterally.

## Definition
A profile \(s^\*\) is a Nash equilibrium if for each player \(i\),
\[
u_i(s_i^\*, s_{-i}^\*) \ge u_i(s_i, s_{-i}^\*) \quad \text{for all } s_i.
\]

In [[topics/prisoners-dilemma]], (Defect, Defect) is a Nash equilibrium.
